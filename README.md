# Hightlight-Extension
How to Install
Create a folder on your computer (e.g., RainbowHighlighter).

Save the 6 generated files inside that folder using the exact filenames specified (e.g., manifest.json, content.js).

Open Google Chrome and go to chrome://extensions.

Toggle Developer mode in the top right corner.

Click Load unpacked.

Select the folder you created.

Open any webpage, select some text, and the rainbow menu will appear!
