# Hightlight-Extension
How to Install

Step 1: Create a folder on your computer (e.g., RainbowHighlighter).

Step 2: Save the 6 generated files inside that folder using the exact filenames specified (e.g., manifest.json, content.js).

Step 3: Open Google Chrome and go to chrome://extensions.

Step 4: Toggle Developer mode in the top right corner.

Step 5: Click Load unpacked.

Step 6: Select the folder you created.

Step 7: Open any webpage, select some text, and the rainbow menu will appear!
